隐私政策

生效日期：2026-07-22

应用名称：data-app（以下简称“本应用”）

运营主体：wind

联系邮箱：zhengjingyi1@foxmail.com

1. 概述

本应用通过 TikTok for Developers 提供的 Login Kit 与 Display API，在获得 TikTok 用户明确授权后，读取该用户自有 TikTok 企业号的部分公开数据，用于企业内部数据分析与经营概览展示。

2. 我们获取的数据

  • 账号基础信息：昵称、头像、Open ID（经 user.info.basic 获取）
  • 账号概览指标：粉丝数、关注数、获赞总数、视频数（经 user.info.stats 获取）
  • 视频列表与元数据：已发布视频标题、封面、发布时间、播放/点赞/评论/分享数（经 video.list 与 video.insights 获取）

我们不收集用户的私信、登录密码、手机号、精确地理位置，也不收集 TikTok 未通过授权接口返回的受众画像数据。

3. 数据使用目的

  • 在企业内部 BI 系统中展示账号概览与视频表现
  • 计算播放量趋势、互动率等聚合指标
  • 排查接口异常与数据对账

4. 数据存储与安全

授权获取的 access_token 与 refresh_token 加密存储于服务端环境变量或密钥管理中，不在前端明文暴露。视频元数据仅保留最近 365 天，定期清理。服务器访问受 IP 白名单与最小权限控制。

5. 数据共享

我们不会将 TikTok 用户数据出售给第三方。仅限授权企业员工在内网系统查看，不对外公开。

6. 用户权利

用户可随时在 TikTok App 的「设置 → 隐私 → 管理应用授权」中取消对本应用的授权，取消后我们将停止拉取并删除该 Open ID 下的缓存数据。

7. 合规

本政策参照 GDPR、CCPA 及 TikTok Developer Guidelines 编写。如有疑问请联系 zhengjingyi1@foxmail.com。
